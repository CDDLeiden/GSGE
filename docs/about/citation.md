## Acknowledging GSGE

When mentioning GSGE in your paper, you can use:

> "We used GSGE (Group-SELFIES Graph Embeddings) [citation] for fragment-based molecular representation and embedding generation."

Or for methods section:

> "Molecular fragments were extracted and embedded using GSGE [citation], a fragment-based representation learning framework that combines group-SELFIES tokenization with graph neural network embeddings. Fragments were extracted from [dataset] using custom bond-cutting rules, and a graph autoencoder with AttentiveFP architecture was trained to generate continuous embeddings."

## Example Applications

### Molecular Property Prediction

> "Fragment embeddings generated with GSGE were used as node features in our molecular graph neural network for property prediction."

### Molecular Generation

> "We tokenized molecules using GSGE's fragment-based vocabulary and trained a transformer model on the resulting token sequences."

### Chemical Space Analysis

> "Chemical space visualization was performed using t-SNE on fragment embeddings generated by GSGE's graph autoencoder."

### Virtual Screening

> "Compound graphs were created using GSGE and compared via graph similarity metrics for virtual screening."
