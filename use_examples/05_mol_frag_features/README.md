# Molecular Fragment Descriptors

This tutorial demonstrates how to calculate and use RDKit molecular descriptors for fragments.

| Tutorial | Time Category | Time Estimate | Difficulty | Description |

## Overview

Fragment descriptors provide chemical and physical properties of molecular fragments:
- **2D Descriptors**: Topological and connectivity features (MW, TPSA, H-bond donors/acceptors)
- **3D Descriptors**: Geometric features (requires 3D conformers)
- **Combined Features**: Descriptors + learned embeddings for enhanced representation

## Quick Start

### Calculate 2D Descriptors

```python
from GSGE import GSGE

# Load GSGE with vocabulary
gsge = GSGE(GS_vocab='vocab.pkl')

# Calculate descriptors for all fragments
gsge.calc_fragment_descriptors(
    descriptor_keys=[
        'MolWt',           # Molecular weight
        'TPSA',            # Topological polar surface area
        'NumHDonors',      # H-bond donors
        'NumHAcceptors',   # H-bond acceptors
        'NumRotatableBonds',  # Rotatable bonds
        'NumAromaticRings',   # Aromatic rings
        'NumHeteroatoms'      # Heteroatoms
    ]
)

# Get descriptor matrix
descriptors = gsge.get_fragment_descriptors()
print(descriptors.shape)  # [num_fragments, num_descriptors]
```

### Load Pre-computed Descriptors

```python
from GSGE import GSGE, get_tests_dir
```

## Available Descriptors

### Common 2D Descriptors

| Descriptor | Description | Use Case |
|------------|-------------|----------|
| `MolWt` | Molecular weight | Size proxy |
| `TPSA` | Topological polar surface area | Membrane permeability |
| `NumHDonors` | H-bond donors | Solubility |
| `NumHAcceptors` | H-bond acceptors | Solubility |
| `NumRotatableBonds` | Rotatable bonds | Flexibility |
| `NumAromaticRings` | Aromatic rings | Stability |
| `NumHeteroatoms` | Heteroatoms (N, O, S, etc.) | Reactivity |
| `Chi0`, `Chi1` | Connectivity indices | Topology |
| `BalabanJ` | Balaban J index | Branching |
| `LabuteASA` | Labute accessible surface area | Solvent exposure |

### 3D Descriptors

For 3D descriptors, conformers are required:

```python
# 3D descriptors need conformers
gsge.calc_fragment_descriptors(
    descriptor_keys=[
        'PMI1', 'PMI2', 'PMI3',  # Principal moments of inertia
        'RadiusOfGyration',       # Shape descriptor
        'Asphericity',            # Deviation from spherical
        'Eccentricity',           # Shape eccentricity
        'SpherocityIndex'         # Spherical character
    ],
    generate_conformers=True  # Auto-generate conformers
)
```

## Descriptor Normalization

Descriptors are automatically normalized using z-score normalization:

```python
# Normalization happens internally
gsge.calc_fragment_descriptors(descriptor_keys=keys)

# Access normalized descriptors
normalized = gsge.get_fragment_descriptors()

# Access normalization parameters
means = gsge.descriptor_calculator.descriptor_means
stds = gsge.descriptor_calculator.descriptor_stds
```

### Manual Normalization

```python
from GSGE.fragment_descriptors import normalize_descriptors, denormalize_descriptors
import numpy as np

# Raw descriptors
raw_desc = np.array([[46.07, 20.23, 1, 1],
                     [78.11, 40.46, 2, 2]])

# Normalize
normalized, means, stds, mask = normalize_descriptors(raw_desc)

# Denormalize
denormalized = denormalize_descriptors(normalized, means, stds)
```

## Combining Descriptors with Embeddings

Combine RDKit descriptors with learned embeddings:

```python
# Calculate both
gsge.calc_fragment_descriptors(descriptor_keys=['MolWt', 'TPSA', 'NumHDonors'])
gsge.make_GS_fragment_embedding_dict()  # After GAE training

# Get combined features
combined = gsge.get_fragment_descriptors_and_embeddings()
print(combined.shape)  # [num_fragments, emb_dim + desc_dim]
```

## Use Cases

### Property Prediction

```python
from sklearn.ensemble import RandomForestRegressor
from sklearn.model_selection import train_test_split

# Get fragment descriptors as features
descriptors = gsge.get_fragment_descriptors()

# Use as molecular features (average fragment descriptors)
def get_mol_descriptors(smiles, gsge, desc_matrix):
    """Get average fragment descriptors for molecule."""
    tokens = gsge.preprocess_from_SMILES(smiles)
    frag_tokens = [t for t in tokens if t.startswith('GS_frag_')]
    token_dict = gsge.tokenizer.token_dict
    frag_ids = [token_dict[t] for t in frag_tokens]

    mol_desc = desc_matrix[frag_ids].mean(axis=0)
    return mol_desc

# Build feature matrix
X = np.array([get_mol_descriptors(s, gsge, descriptors) for s in smiles_list])
y = property_values

# Train model
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2)
model = RandomForestRegressor()
model.fit(X_train, y_train)

# Evaluate
score = model.score(X_test, y_test)
print(f"R² Score: {score:.3f}")
```

### Fragment Similarity

Compare fragments using descriptor similarity:

```python
from scipy.spatial.distance import cosine

# Get descriptors for two fragments
desc1 = descriptors[0]  # First fragment
desc2 = descriptors[5]  # Fifth fragment

# Calculate similarity
similarity = 1 - cosine(desc1, desc2)
print(f"Descriptor similarity: {similarity:.3f}")
```

### Clustering Fragments

```python

from sklearn.manifold import TSNE
import matplotlib.pyplot as plt

# Cluster fragments by descriptors
kmeans = KMeans(n_clusters=10, random_state=42)
clusters = kmeans.fit_predict(descriptors)

# Visualize
tsne = TSNE(n_components=2, random_state=42)

plt.scatter(tsne_result[:, 0], tsne_result[:, 1], c=clusters, cmap='tab10')
plt.colorbar(label='Cluster')
plt.xlabel('t-SNE 1')
plt.ylabel('t-SNE 2')
plt.title('Fragment Clustering by Descriptors')

```

### Feature Selection

Identify most informative descriptors:

```python
from sklearn.feature_selection import SelectKBest, f_regression

# Select top K descriptors
```


## Performance Tips

1. **Cache Descriptors**: Calculate once and save
2. **Select Relevant Descriptors**: Don't compute unnecessary descriptors
3. **Batch Processing**: RDKit is optimized for batch calculations
4. **3D Conformers**: Skip if not needed (computationally expensive)

## Troubleshooting

### Descriptor Calculation Fails

**Problem**: Cannot calculate descriptors for some fragments

**Solutions**:
- Check fragment SMILES validity
- Skip problematic fragments
- Use try-except for robust calculation

### NaN Values

**Problem**: Descriptors contain NaN

**Solutions**:
- Check for undefined descriptors (e.g., 3D without conformers)
- Use `fillna()` to handle missing values
- Remove features with too many NaN

### Memory Issues

**Problem**: Out of memory with large vocabularies

**Solutions**:
- Calculate descriptors in batches
- Select subset of descriptors
- Use sparse representations if applicable

## Best Practices

1. **Choose Relevant Descriptors**: Select descriptors relevant to your task
2. **Normalize**: Always normalize before ML
3. **Check Correlations**: Remove highly correlated descriptors
4. **Combine with Embeddings**: Use both for best performance
5. **Validate**: Check descriptor distributions make chemical sense
